# Tez Cargo – Регистрация

Простое веб-приложение для регистрации пользователей Tez Cargo.

## Описание
Этот проект позволяет пользователям пройти регистрацию с указанием:

- ФИО  
- Телефон (WhatsApp)  
- Город  
- Адрес (необязательно)  

После успешной регистрации отображается клиентский код, а пользователи получают доступ к функционалу бота.

Регистрация работает с Google Sheets через Web App и поддерживает Telegram WebApp API.

## Файлы проекта
- `registration.html` — главная страница с формой регистрации  
- `.gitignore` — исключения для Git  
- `README.md` — описание проекта
